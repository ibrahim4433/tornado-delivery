# Tech Stack & Constraints

## Core Frameworks
- **Runtime:** Browser (Static HTML/CSS/JS).
- **Hosting:** GitHub Pages.
- **Language:** HTML5, CSS3, Vanilla JavaScript (ES6+).

## UI & Styling
- **CSS Engine:** Native CSS3 (Variables used for theming).
- **Design System:** Mobile-First Responsive Design.
- **Icons:** Emojis (current) or SVG Icons (preferred for speed).
- **Brand Colors:**
  - Primary Orange: `#f36f21`
  - Background Dark: `#1a1a1a`
  - Text White: `#ffffff`

## Infrastructure
- **Build Tool:** None (Raw static files).
- **Package Manager:** None (Do not use npm/yarn).

## STRICT PROHIBITIONS
- **DO NOT** use React, Vue, or Angular (Keep it lightweight).
- **DO NOT** use Bootstrap or heavy CSS frameworks.
- **DO NOT** introduce a build step (Webpack/Vite) unless explicitly authorized.
